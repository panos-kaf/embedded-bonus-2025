# Dynamic Memory Management Bonus Project

This repository contains the source code for the Strategic Dynamic Memory Management bonus project of the Embedded Systems NTUA course.

### The project is organized into two main directories:

*   **[Heap-Layers](Heap-Layers/)**: Contains modifications and extensions to the Heap-Layers framework for custom memory allocator implementation.
*   **[mimalloc-bench](mimalloc-bench/)**: Contains the benchmarking infrastructure based on mimalloc-bench to evaluate the performance of our allocators.
